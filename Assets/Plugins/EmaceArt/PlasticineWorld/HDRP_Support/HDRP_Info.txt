Open the project in common Unity 2018.02 or Unity 2019.03 + HDRP

Instruction for Unity 2019 + HDRP

Run Unity with ready profile HDRP (the best with the use of Unity HUB). Download Plasticine World from the Asset Store. Look for the folder Emaceart>Assets>PlasticineWorld>HDRP_Support and click Package file. After longer time, when imstallation is finished, you should see correctly the scene with ready to use project in the HDRP standard.

Unfortunately, at the moment grass shader doesn`t work in Unity 2019 HDRP

Otwórz projekt w zwykłej wersji Unity 2018.02 lub Unity 2019.03 + HDRP

Instrukcja dla Unity 2019.03 + HDRP. 

Uruchom Unity z gotowym profilem HDRP (najlepiej z użycie Unity HUB)
Ściągnij necropoly z Asset Store. Zajrzyj do folderu Emaceart>Assets>HDRP_Support i kliknij w plik package tam zawartym. Po dłuższym czasie po tym jak zakończy się instalacja, powinieneś zobaczyć poprawnie wyświetlaną scenę i gotowym do użycia projektem w standardzie HDRP.

Nie stety na dzień dzisiejszy, Unity z standardem HDRP nie wspiera shaderów z obsługą wiatru.
